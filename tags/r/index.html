Scraping NBA Stats From The Web

Web scraping using R is a fairly easy task. Let’s take a look at the Coach of the Year statistics from the ESPN.com site. First install the necessary packages. rvest allows you to extract data from a webpage. stringr allows you to manipulate strings. tidyr will load the data manipulation libraries that’ll be useful for selecting and munging data. install.packages('rvest') install.packages('stringr') install.packages('tidyr') We add an extra library here because html_nodes() function fails when not run interactively.
